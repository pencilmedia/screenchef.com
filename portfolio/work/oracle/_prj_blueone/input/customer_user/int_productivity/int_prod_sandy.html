HAND DATA ENTRY CASES

1. Describe the process you follow to enter PARTS by hand. What works well in Agile? What about Agile makes this difficult?

Collect data sheet.  Initiate NPR, fill in data from datasheet.  Attach docs and submit to process.  Doc Control returns a real IPN.


2. Describe the process you follow to enter / modify BOMs by hand. What works well in Agile? What about Agile makes this difficult?

Tellabs Dallas uses a PECO process for early BOM changes.  Lightweight signatures that allows for very fast BOM changes during design.  Often get missorted Ref Des - not in numerical order.  Use design status to indicate part quality.  When interting parts by hand, set the find number manually to insert.  Use Find Numbers 100, 110, 120, etc that allows for inserts.  .

3. Describe your change management process? How are BOMs, AMLs, Drawings changed during the ECO process? What works well in Agile? What about Agile makes this difficult?

Signoff can take weeks.  8 to 10 signatures.  Would really want pesteing emails.  Some process analysis issues?
