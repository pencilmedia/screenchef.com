How to Style the Coqsoft Table

Follow these simple instructions to make the Coqsoft table conform to Agile's BlueOne look and feel. Style changes are confined to three files: agile.gif, agile.css and your table's XML configuration file.

  • agile.gif - By default, Coqsoft includes Grid.gif to retrieve the icons that make up its toolbar. Agile.gif is a modification of Grid.gif that removes the ugly icons for row selection and deletion.
  • agile.css - This CSS overrides some of the styles defined in Coqsoft's required Grid.css file. This file must be included after Grid.css.
  • XML configuration file - This is the XML configuration required by every Coqsoft table. You'll need to copy and paste a snippet of configuration code to your table's XML file.

Table.html and table.xml are provided as an example on how to stylize the Coqsoft table. You'll need to follow these 3 steps:

  1. Verify that agile.css and agile.gif are in Coqsoft's Grid folder (the folder containing Grid.css). If the files aren't there, copy them into this folder.
  2. Copy and paste the XML snippet delimited by the BEGIN and END comments to your XML configuration file. The XML here will do the job of setting colors, hiding the bottom toolbar, removing icons, etc.
  3. In your HTML file, link to agile.css after linking to Grid.css.

That's it! These instructions represent the current state of the Coqsoft table's BlueOne makeover. I'll be updating these steps as I continue to make progress.
