Using Automatic Upload

Close

The Automatic Upload feature provides the ability for you to automate the process of loading new data sets into Vendavo Profit Advisor. Regardless of your data's source system, you can run your extraction process on your source system and save the data output to the Vendavo-created FTP folder.

When using Automatic Upload, keep the following details in mind:

  • Requires credentials for accessing the Vendavo-created FTP folder. The URL, username, and password were emailed to your Customer Account Owner.
  • The Vendavo Profit Advisor application checks the FTP folder every 30 minutes and uploads any new files without the opportunity to review data issues, if any occur.
  • It is recommended that no more than 25 files be added at a time to the FTP folder; zipping those files will improve upload time.
  • The analysis of the files is performed at the frequency established in the Data Settings. The default frequency is weekly. You can change this setting by clicking the gear icon at the top right of the Data workspace.
